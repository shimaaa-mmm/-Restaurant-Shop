## ✨ درباره‌ی پروژه

**رستوران ما** یک وب‌اپلیکیشن کامل فروشگاهی است که با هویت بصری الهام‌گرفته از طعم و رنگ **زعفران ایرانی** طراحی شده. این پروژه با داده‌های واقعی از [TheMealDB](https://www.themealdb.com/api.php) کار می‌کند و تجربه‌ای شبیه به یک فروشگاه واقعی — از جست‌وجوی زنده تا پرداخت آنلاین — را شبیه‌سازی می‌کند.

> ساخته‌شده برای نمایش در رزومه و پورتفولیو 💼 — با معماری تمیز، کد قابل‌توسعه و جزئیات حرفه‌ای در UI/UX.

---

## 🚀 قابلیت‌ها

| دسته | امکانات |
|---|---|
| 🍛 **منو و محصولات** | اتصال زنده به TheMealDB، جست‌وجوی زنده (debounce)، فیلتر بر اساس دسته و کشور، غذای تصادفی |
| 🖼️ **کارت‌های حرفه‌ای** | امتیاز ستاره‌ای، تعداد نظرات، برچسب «پرطرفدار»، مودال مشاهده‌ی سریع |
| ❤️ **علاقه‌مندی‌ها** | افزودن/حذف با یک کلیک، ذخیره در دستگاه کاربر |
| 🛒 **سبد خرید** | افزایش/کاهش تعداد، حذف با انیمیشن، محاسبه‌ی خودکار جمع کل |
| 💳 **پرداخت** | چک‌اوت دو‌مرحله‌ای، اعتبارسنجی شماره کارت با الگوریتم Luhn، تشخیص بانک، پرداخت در محل |
| 📊 **داشبورد کاربر** | تاریخچه‌ی سفارش‌ها، نمودار خرید، آمار کلی، مدیریت علاقه‌مندی‌ها |
| 🔐 **احراز هویت** | ثبت‌نام و ورود با اعتبارسنجی، مسیرهای محافظت‌شده (Private Routes) |
| 🎨 **UI/UX** | راست‌به‌چپ (RTL) کامل، فونت وزیرمتن، انیمیشن‌های Framer Motion، پارالاکس، اسلایدر، اسکلتون لودینگ |
| 🌐 **پایداری API** | تلاش مجدد خودکار (retry) و کش محلی برای دسترس‌پذیری همیشگی |

## 🛠️ پشته‌ی فناوری

- **React 18** + **Vite 5** — رابط کاربری سریع و مدرن
- **Redux Toolkit** — مدیریت state (سبد خرید، کاربر، غذاها، علاقه‌مندی‌ها)
- **React Router v6** — مسیریابی
- **Tailwind CSS 3.4** — استایل‌دهی utility-first
- **Framer Motion** — انیمیشن‌های نرم و حرفه‌ای
- **MUI + Ant Design** — کامپوننت‌های تعاملی (مودال، پاپ‌اور)
- **Axios** — ارتباط با API

## 📦 نصب و اجرا

```bash
# ۱. نصب پکیج‌ها
npm install

# ۲. اجرای پروژه در حالت توسعه
npm run dev

# ۳. ساخت نسخه‌ی نهایی برای انتشار
npm run build
```

پروژه روی آدرس `http://localhost:5173` بالا می‌آید.

> 💡 اگر پیام خطای مربوط به `rollup` یا بسته‌های native دریافت کردید، کافی‌ست `node_modules` و `package-lock.json` را حذف و دوباره `npm install` را اجرا کنید.

## 📁 ساختار پروژه

```
src/
├── api/           # ارتباط با TheMealDB و سرویس پرداخت
├── app/           # پیکربندی Redux store
├── components/    # کامپوننت‌های قابل‌استفاده‌ی مجدد (هدر، مودال، اسلایدر...)
├── features/      # اسلایس‌های Redux (auth, cart, meals, wishlist)
├── pages/         # صفحات اصلی (منو، جزئیات، سبد خرید، چک‌اوت، داشبورد...)
└── utils/         # توابع کمکی (قیمت‌گذاری، امتیاز، تاریخچه‌ی سفارش)
```

این پروژه از **[TheMealDB](https://www.themealdb.com/api.php)** به‌عنوان منبع داده‌ی غذاها استفاده می‌کند (رایگان و بدون نیاز به کلید API برای سطح پایه).
